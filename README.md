GRAMBEE — Платёжная система, финансовая аналитика и закупка трафика в Telegram
Платформа для управления финансами, аналитики и автоматизированной закупки трафика в экосистеме Telegram. Включает платёжный модуль с поддержкой Tinkoff и USDT, AI-чат, финансовую аналитику и форму создания рекламных кампаний.

🚀 Возможности
💳 Платёжная система — пополнение баланса через Tinkoff и USDT с автоматическим обновлением статуса

📊 Финансовая аналитика — графики движения средств, таблица операций с виртуальным скроллингом

🤖 AI-чат — интеллектуальный ассистент с сохранением истории и быстрыми командами

📢 Закупка трафика — создание рекламных кампаний в Telegram-каналах/чатах/ботах за 2–3 минуты

🔐 Telegram Mini Apps — нативная авторизация через initDataUnsafe

⚡ Автообновление данных — рефетчинг баланса и курсов при фокусе окна

🛠 Технологический стек
React — интерфейс пользователя

TypeScript — типобезопасность

Zustand — управление состоянием (пользователь, баланс, статусы платежей)

TanStack Query — кеширование и синхронизация данных с сервером

Styled Components — стилизация компонентов

SVG — кастомные графики аналитики

Telegram Mini Apps SDK — интеграция с экосистемой Telegram

Vite — сборка проекта

CI/CD — автоматический деплой

🧩 Архитектура и реализация
Платёжный модуль и финансовая аналитика
Спроектирована архитектура платёжного модуля с интеграцией Telegram Mini Apps SDK и платёжных шлюзов (Tinkoff, USDT). Реализована защита от дублирования платежей и автоматическое обновление статуса.

Результат:

Безопасные и надёжные транзакции

Минимизация ошибок при пополнении баланса

Управление состоянием
Внедрён Zustand для локального состояния (пользователь, баланс, статусы платежей). Использован TanStack Query для кеширования баланса и курсов валют с автоматическим рефетчингом.

Результат:

Снижение нагрузки на сервер

Мгновенное обновление данных

Актуальная информация без ручного обновления страницы

AI-чат
Разработан интеллектуальный чат с сохранением истории последних 50 сообщений, автоскроллом и системой быстрых команд. Пользователь выполняет действия с 1 клика вместо 3–4.

Результат:

Ускорение взаимодействия с платформой

Повышение удобства использования

Снижение когнитивной нагрузки на пользователя

Финансовая аналитика
Реализован график движения средств на SVG и таблица операций с виртуальным скроллингом для работы с большими объёмами данных.

Результат:

Наглядная визуализация финансовых потоков

Плавная работа даже при тысячах транзакций

Быстрый поиск и фильтрация операций

Закупка трафика (ключевая фича)
Разработана форма создания рекламных кампаний с таргетингом (страны, дни, бюджет, возраст). Реализована валидация пересечений и проверка канала по ссылке/токену — без проверки кампания не создаётся. Добавлено сохранение черновиков в localStorage.

Было: настройка кампании занимала 15–20 минут вручную
Стало: 2–3 минуты на полную настройку

Результат:

Сокращение времени на создание кампании на 80–85%

Исключение ошибок при вводе данных

Автосохранение прогресса

Безопасность и валидация
Чек на лету, проверка каналов по ссылке/токену, защита от дублирования платежей и кампаний.

Результат:

Надёжная защита от мошенничества

Валидные данные на всех этапах

Стабильная работа платформы
